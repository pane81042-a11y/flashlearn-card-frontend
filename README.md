# FlashLearn Card Frontend

FlashLearn Card Frontend handles flashcard level management within each deck.

It integrates with the Card Service for storage and retrieval.

## Features

- Create flashcards
- Edit front and back content
- Delete flashcards
- Bulk card management
- API integration with Card Service

## Architecture

- React
- Webpack Module Federation
- Axios
- Shared UI library

Loaded dynamically by FlashLearn Shell.

## AWS Deployment

- Amazon S3
- Amazon CloudFront
- GitHub Actions CI/CD

## Portfolio Value

Demonstrates:

- Modular microfrontend architecture
- Scalable card management system
- Serverless frontend hosting
